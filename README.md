# 🛒 Blinkit Grocery Sales Analysis (Excel Dashboard)

This project presents an interactive **Excel-based dashboard** analyzing sales data from **Blinkit**, India's last-minute grocery delivery app. The dashboard provides insights into outlet performance, item types, fat content, and geographic tiers.

## 🧩 Features & Visuals

- **🎛️ Interactive Filter Panel**:
  - Outlet Size: High, Medium, Small  
  - Outlet Location: Tier 1, Tier 2, Tier 3  
  - Item Type: Multiple categories  

- **📊 Charts Included**:
  - Donut Charts (Fat Content, Outlet Size)  
  - Bar Charts (Item Type, Outlet Location, Outlet Type)  
  - Line Chart (Outlet Establishment Trend)  
  - KPI Cards (Total Sales, Avg Sales, Items, Rating)  

- **⚡ Dynamic Dashboard**  
  - Updates automatically based on filter selection  

---

## 🧠 Insights Uncovered

- Regular fat items contribute around **65% of total sales**  
- **Tier 3 outlets** generate the highest sales  
- Top categories include **Fruits & Vegetables, Snack Foods, Household Items**  
- Sales peak observed in **2018**

## 📌 How to Use

1. Open `Blinkit Grocery Data Analysis.xlsx`  
2. Use the **Filter Panel** on the left side  
3. Explore different insights using charts  
4. Click on **Dataset Button** to view raw data
